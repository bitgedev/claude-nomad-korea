# Nomad Korea — 단계별 구현 계획 (SPEC)

## 현재 상태 요약

- 랜딩 페이지 전용: Hero, StatsBar, TopCities, Features, Activity, Reviews, CtaBanner 섹션으로 구성
- Navbar 링크: 모두 `href="#"` — 실제 라우팅 없음
- 검색창: UI만 존재, 기능 없음
- 내부 페이지 없음: `/login`, `/signup`만 존재 (Supabase 연결됨)
- Mock 데이터: 3개 도시, 3개 리뷰, 5명 노마드, 5개 밋업
- 스택: Next.js 15 App Router, TypeScript, Tailwind CSS v4, Shadcn UI (base-ui 기반, `asChild` 불가)

---

## 단계별 의존성

```
Phase 1 (라우팅 + Mock 데이터)
    ├── Phase 2 (도시 페이지)
    ├── Phase 3 (밋업 페이지)
    ├── Phase 4 (리뷰 페이지) ← Phase 2 선행 권장
    └── Phase 5 (인증)
            └── Phase 6 (코워킹 + UX 완성) ← 모든 Phase 완료 후
```

Phase 2, 3, 5는 Phase 1 완료 후 **병렬 진행 가능**. Phase 4는 Phase 2 완료 후 진행 권장.
