## 시작하기

```bash
cp .env.example .env.local  # 환경변수 설정
npm install && npm run dev
```

## 환경변수

```
NEXT_PUBLIC_SITE_URL=https://claude-nomad-korea.vercel.app/
NEXT_PUBLIC_SUPABASE_URL=
NEXT_PUBLIC_SUPABASE_PUBLISHABLE_KEY=
```
